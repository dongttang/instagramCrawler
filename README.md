### instagram picture crawler
- 심심해서 발로짠 인스타 사진 수집기
- 특정 유저가 게시한 모든 사진을 한번에 다운로드 해줍니다
#### requirement
- MacOS
- python3.x
- BeautifulSoup4
- selenium
- wget
- webDriver(used chromedriver here)
- chrome

#### usage
1. 라이브러리 및 웹드라이버 설치.(크롬드라이버 사용)
2. 스크립트 파일을 실행하면 사진을 수집할 인스타그램 계정 아이디, 저장할 path, 웹 드라이버 path를 물어봄.
3. 입력하고 엔터누르면 끄읏.
